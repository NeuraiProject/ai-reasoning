# Neurai Reasoning with DeepSeek-R1

## Getting Started

Follow the steps below to set up and run the application.

### 1. Clone the Repository

Clone the examples repository from GitHub:

```sh
git clone https://github.com/neuraiproject/ai-reasoning
```

### 2. Navigate to the Project Directory

Change your working directory to the `ai-reasoning/deepseek-r1` folder:

```sh
cd ai-reasoning/deepseek-r1
```

### 3. Install Dependencies

Install the necessary dependencies using npm:

```sh
npm install
```

### 4. Run the Development Server

Start the development server:

```sh
npm run dev
```


### 5. Build HTML version

Start the build version:

```sh
npm run build
```

The application should now be running locally. Open your browser and go to `http://localhost:19888` to see it in action.
